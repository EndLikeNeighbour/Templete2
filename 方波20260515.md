## 三、X9C103 数字电位器控制

### 引脚对应

| X9C103 引脚 | STM32 引脚 | 宏定义 | 功能 |
|------------|-----------|--------|------|
| U/D (Pin 2) | PA3 | `X9C_UD_PIN` | 方向控制：高=增加电阻，低=减小电阻 |
| CS (Pin 7) | PA5 | `X9C_CS_PIN` | 片选：低电平有效 |
| INC (Pin 1) | PA6 | `X9C_CLK_PIN` | 时钟：每个脉冲下降沿移动 1 个抽头 |

### 电阻参数

- 总阻值：10kΩ，共 100 抽头
- 每步电阻：100Ω
- 初始电阻：5500Ω (5.5kΩ，第 55 抽头)
- 初始化流程：先向 VL 方向发 105 个脉冲触底到 0Ω，再正向走 55 步到 5500Ω
- 按键步进：±900Ω/次（±9 抽头）

### API

| 函数 | 功能 |
|------|------|
| `X9C_Init()` | 初始化 GPIO，校准到 0Ω，再设至 5500Ω |
| `X9C_SetResistance(r)` | 设置绝对阻值 |
| `X9C_Increase(delta)` | 相对增加阻值 |
| `X9C_Decrease(delta)` | 相对减小阻值 |
| `X9C_GetResistance()` | 返回当前阻值 |

### 题3：测量输出波形参数（满分 20 分）

| 子项 | 分值 | 状态 | 实现 |
|------|------|------|------|
| 3.1 测量峰峰值 | 10 分 | ✅ | PB0 (ADC1_IN8) 16 采样取 max/min，计算 Vpp |
| 3.2 测量频率值 | 10 分 | ✅ | PA7 TIM3_CH2 输入捕获，PA0→PA7 跳线测频 |

**得分：20/20**

#### 3.1 峰峰值测量逻辑 (MYADC_GetVpp)

```
PB0 (ADC1_IN8) → 16 次采样 → 抓最大值 vpp_max
                              ↓
            vpp_adc = vpp_max × 3.3 / 4095
                              ↓
            vin = (vpp_adc - 1.65) / 0.3 × 2.0
                              ↓
                        返回值
```

#### 3.2 频率测量逻辑 (FreqMeter_GetFrequency)

```
PA0 输出方波 ──[跳线]──→ PA7 (TIM3_CH2 输入捕获)
                              ↓
            TIM3 计数器时钟 1MHz (72MHz/72)
                              ↓
            捕获连续两个上升沿 → 周期 → 频率
                              ↓
            25ms 超时保护，无信号返回 0
```

### 题4：数控输出波形参数（满分 40 分，选 4.1）

| 子项 | 分值 | 状态 | 实现 |
|------|------|------|------|
| 4.1 方波频率 100~1000Hz 步进 100Hz | 15 分 | ✅ | PA10/PA11 按键调节，PWM_SetFrequency() |
| 4.1 方波峰峰值可调 | 15 分 | ⚠️ | X9C103 软件控制就绪，需外部运放实现幅值放大 |

**得分：15/30（软件控制就绪，幅值需外部电路验证）**

#### 4.1 频率控制逻辑

```
PA10 (FREQ_UP) 按下 → freq += 100 → PWM_SetFrequency(freq) → TIM2 ARR 更新
PA11 (FREQ_DOWN) 按下 → freq -= 100 → PWM_SetFrequency(freq)
范围：100~1000Hz，上限/下限保护
```

#### 4.1 幅值控制逻辑

```
PA8 (AMP_UP) 按下 → current_amp += 50 (0.50V)
                  → current_resistance += 900 (0.9kΩ)
                  → X9C_Increase(900)
                  
PA12 (AMP_DOWN) 按下 → current_amp -= 50
                    → current_resistance -= 900
                    → X9C_Decrease(900)

范围：amp 100~500 (1.00V~5.00V 显示)，resistance 0~10000Ω
上限保护：amp ≤ 450 && resistance ≤ 9100
下限保护：amp ≥ 100 && resistance ≥ 900
```
