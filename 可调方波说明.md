# 可调方波输出说明（数字电位器版）

## 功能特性

- **幅值可调**：1-10V，步进1V（通过数字电位器实现真正可调）
- **频率可调**：100-1000Hz，步进100Hz（通过旋转编码器）
- **按键控制**：按键控制幅值，编码器控制频率
- **OLED显示**：实时显示参数

## 硬件连接

### STM32引脚分配

| 引脚 | 功能 | 说明 |
|------|------|------|
| PA0 | PWM方波输出 | TIM2_CH1，0-3.3V |
| PA2 | 编码器A相 | 上拉输入 |
| PA3 | 编码器B相 | 上拉输入 |
| PA4 | MCP4131 CS | 数字电位器片选 |
| PA5 | MCP4131 CLK | 数字电位器时钟 |
| PA6 | MCP4131 DI | 数字电位器数据输入 |
| PA7 | 3.3V输出 | 给硬件电路参考 |
| PC0-4 | 按键 | UP/DOWN/LEFT/RIGHT/OK |

### 旋转编码器连接

```
┌─────────────────────────────────────┐
│         旋转编码器 (EC11)           │
├─────────────────────────────────────┤
│  [VCC] ── 3.3V                     │
│  [GND] ── GND                      │
│  [A]   ── PA2 (上拉输入)           │
│  [B]   ── PA3 (上拉输入)           │
│  [SW]  ── 可选（未使用）            │
└─────────────────────────────────────┘
```

### 数字电位器连接 (MCP4131)

```
┌─────────────────────────────────────────────┐
│           MCP4131 (SOP-8)                    │
├─────────────────────────────────────────────┤
│  1 [VDD] ── 3.3V                            │
│  2 [P0W] ── 运放反馈电阻（滑动端）           │
│  3 [P0A] ── 运放输出端                       │
│  4 [P0B] ── 运放反相输入端                   │
│  5 [H0]  ── GND                             │
│  6 [WP]  ── 3.3V (禁止写保护)               │
│  7 [SHDN] ── 3.3V (禁止关断)                 │
│  8 [VSS] ── GND                             │
├─────────────────────────────────────────────┤
│  [CS]  ── PA4                               │
│  [CLK] ── PA5                               │
│  [DI]  ── PA6                               │
└─────────────────────────────────────────────┘
```

### 完整硬件电路框图

```
                    ┌──────────────────────────────────────┐
PA0 (0-3.3V方波) ──→│  减法器/放大器                       │
                    │                                      │
                    │  ┌────────┐    ┌──────────────┐     │
                    │  │ 运放   │←───│MCP4131数字   │     │
                    │  │        │    │电位器        │     │
                    │  └────────┘    └──────────────┘     │
                    │        │                             │
                    └────────┼─────────────────────────────┘
                             │
                             ↓
                        输出 1-10V 可调
                   
PA7 (3.3V) ──[分压1/2]───→ 1.65V ──→ 减法器同相输入端
```

## 控制方式

### 按键控制

| 按键 | 功能 | 说明 |
|------|------|------|
| KEY_UP (PC0) | 幅值+1V | 控制MCP4131增加 |
| KEY_DOWN (PC1) | 幅值-1V | 控制MCP4131减少 |

### 编码器控制

| 操作 | 功能 | 说明 |
|------|------|------|
| 顺时针旋转 | 频率+100Hz | PWM频率增加 |
| 逆时针旋转 | 频率-100Hz | PWM频率减少 |

### OLED显示内容

```
Vpp: 5 V
Freq: 500 Hz
Encoder: 0
```

## 数字电位器工作原理

MCP4131是一个128级数字电位器，通过SPI接口控制：

| 级别 | 电压Vpp | 说明 |
|------|---------|------|
| 13 | 1V | 最小幅值 |
| 64 | 5V | 中间幅值 |
| 127 | 10V | 最大幅值 |

## 编译说明

1. 将新文件添加到工程：
   - `Hardware/mcp4131.c`
   - `Hardware/encoder.c`
   - `Hardware/mcp4131.h`
   - `Hardware/encoder.h`

2. 确保包含路径正确配置

3. 编译烧录到STM32F103C8T6

## 使用说明

1. **硬件连接**：按接线图连接所有元件
2. **上电**：STM32上电后默认幅值5V，频率500Hz
3. **调节幅值**：按KEY_UP/KEY_DOWN按键，1V/步
4. **调节频率**：旋转编码器，100Hz/步
5. **观察输出**：用示波器观察输出波形

## 注意事项

1. MCP4131的供电电压为3.3V，不要接5V
2. 数字电位器需要正确连接到运放反馈回路
3. 旋转编码器需要正确连接A/B相，注意上拉
4. 幅值变化需要通过硬件电路实现，代码只是控制数字电位器
5. 调试时先用示波器观察MCP4131是否正常工作
