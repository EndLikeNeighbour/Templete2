## 三、软件模块功能说明

### 3.1 波形输出（PA0）

| 文件 | 功能 |
|------|------|
| [pwm.c](Hardware/pwm.c) | TIM2_CH1 PWM 输出，频率 100~1000Hz 可调，步进100Hz |
| [pwm.h](Hardware/pwm.h) | `PWM_Init(freq, amp_mv)`、`PWM_SetFrequency(freq)`、`PWM_GetFrequency()` |

- 定时器时钟 72MHz（APB1×2）
- 分频系数 = 72000000/(freq×1000) - 1
- PWM 周期 = 999，占空比 = 50%（固定）
- 默认初始频率：**1000Hz**

### 3.2 幅值控制（PA4~PA6）

| 文件 | 功能 |
|------|------|
| [mcp4131.c](Hardware/mcp4131.c) | MCP4131 数字电位器 SPI 控制，7位分辨率（1~127级） |
| [mcp4131.h](Hardware/mcp4131.h) | `Init/SetLevel/Increase/Decrease/GetLevel/GetVoltageVpp()` |

- 初始等级 = 64
- 按键每按一次 ±13 级（对应约 ±1V 显示）
- `GetVoltageVpp()` = `(level × 10) / 127` 估算峰峰值
- SPI 时序：命令字节 0x00 + 数据字节

### 3.3 直流电压测量（PA1）

| 文件 | 功能 |
|------|------|
| [adc.c](Hardware/adc.c) | ADC1 连续转换，中值+均值+IIR 滤波算法 |
| [adc.h](Hardware/adc.h) | `MYADC_Init()`、`MYADC_GetVoltage()` |

- ADC 时钟 = 12MHz（PCLK2/6）
- 采样周期：239.5 个 ADC 时钟周期
- 预热：初始化时丢弃前 64 次转换
- 读数：取 32 个样本 → 中值×3 + 均值×1 → IIR 滤波 `(fil×15 + raw×1)/16`
- 返回电压 = `(滤波值 × 3.3V) / 4095`

### 3.4 频率测量（PA7）

| 文件 | 功能 |
|------|------|
| [freqmeter.c](Hardware/freqmeter.c) | TIM3_CH2 输入捕获测频 |
| [freqmeter.h](Hardware/freqmeter.h) | `FreqMeter_Init()`、`FreqMeter_GetFrequency()` |

- 计数器时钟 = 1MHz（72MHz / 72）
- 等待连续两个上升沿，测量周期，返回频率
- 25ms 超时保护，无信号返回 0
- **硬件要求**：PA0 → PA7 跳线连接

### 3.5 按键输入（PA8~PA12）

| 文件 | 功能 |
|------|------|
| [key.c](Hardware/key.c) | 5 个按键上拉输入，消抖扫描 |
| [key.h](Hardware/key.h) | `KEY_Init()`、`KEY_Scan()`，返回 `key_state` 枚举 |

- 消抖时间：~1.4ms 每次（Key_DelayMs(5) @ 72MHz）
- 按键按下等待释放后才返回

### 3.6 OLED 显示（PB6~PB9）

| 文件 | 功能 |
|------|------|
| [oled.c](Hardware/oled.c) | SSD1306 128×64 I2C 驱动，软件模拟 I2C |
| [oled.h](Hardware/oled.h) | 显示函数：ShowChar/ShowString/ShowNum/ShowChinese |

- 帧缓冲：`oled_buffer[128][8]`（1024 字节）
- ASCII 字库：8×16 像素，95 个可打印字符
- 中文含 17 个汉字：幅值频率直流电压按键次数：

### 3.7 字库索引

| 文件 | 功能 |
|------|------|
| [font_index.h](Hardware/font_index.h) | 中文字符 `#define` 常量和 ASCII 别名 |

```c
CH_FU(0)  CH_ZHI(1)     // 幅值
CH_PIN(2) CH_LV(3)      // 频率
CH_AN(4)  CH_JIAN(5)    // 按键
CH_CI(6)  CH_SHU(7)     // 次数
CH_ZHI2(8) CH_LIU(9)    // 直流
CH_DIAN(10) CH_YA(11)   // 电压
CH_COLON(12)            // ：
```

```

┌──────────────────────────────────────────────────┐
│ 幅值：05V                ← y=0 (第1行)  MCP4131   │
```

## 七、建议硬件接线

```
                     ┌─────────────────────┐
                     │  STM32F103C8        │
                     │                     │
  示波器/万用表 ←── PA0│  PWM输出             │
  直流电压输入 ───→ PA1│  ADC输入（AIN）       │
  PA0→PA7 跳线 ─── PA7│  测频输入             │
                     │                     │
  MCP4131 CS ──── PA4│                     │
  MCP4131 CLK ─── PA5│  SPI 数字电位器       │
  MCP4131 DI ──── PA6│                     │
                     │                     │
  AMP_UP ──────── PA8│                     │
  AMP_DOWN ────── PA9│  按键                │
  FREQ_UP ────── PA10│  (对GND, 内部上拉)    │
  FREQ_DOWN ──── PA11│                     │
  MEASURE ────── PA12│                     │
                     │                     │
  OLED RST ───── PB6 │                     │
  OLED SCL ───── PB8 │  I2C OLED            │
  OLED SDA ───── PB9 │                     │
                     │                     │
  SWDIO ──────── PA13│  调试 (不可占用)      │
  SWCLK ──────── PA14│                     │
                     └─────────────────────┘
```

> **注意**：频率测量需要 PA0→PA7 外部跳线。PA1 接待测直流电压（0~3.3V）。MCP4131 的数字电位器输出需外部运放才能将 0~3.3V 的 PWM 放大到 1~10V。
